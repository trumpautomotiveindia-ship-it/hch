HCH Electric is a premium dark-themed WooCommerce storefront designed for EV spare-part retailers. It ships with a sticky category navigation, brand attribute filter chips, an off-canvas cart drawer, AJAX add-to-cart and a WhatsApp checkout button built for Indian B2B commerce.

== Description ==

Key features:

* 100 % Block Editor compatible — every section of the homepage is registered as a Gutenberg pattern.
* WooCommerce-ready with template overrides for archive-product, single-product and content-product.
* Brand filter rail powered by the `pa_brand` product attribute.
* Category bar powered by `product_cat` (with optional emoji icon per category).
* Custom meta box on the Product edit screen: Spec line, MOQ, Badge and Emoji fallback icon.
* Off-canvas cart drawer with AJAX qty updates and a one-click "Order via WhatsApp" button (number configurable in the Customizer).
* Full Customizer panel for ticker lines, hero headline, stats, footer text and WhatsApp number.
* Dark UI with cyan / green accents, Syne + Outfit + DM Mono typography.

== Installation ==

1. Upload the `hch-electric` folder to `/wp-content/themes/` OR upload the zip via Appearance → Themes → Add New → Upload Theme.
2. Activate through the Themes screen.
3. Install & activate **WooCommerce**.
4. (Recommended) Products → Attributes → add attribute named **Brand** (slug `brand`) and add terms like OLA S1, Ather 450, Bajaj Chetak, etc. Enable archives.
5. Products → Categories → add your categories and set an emoji icon for each.
6. Appearance → Customize → **HCH Electric** panel to edit ticker / hero / stats / WhatsApp number.
